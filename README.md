# 音訊轉文字與優化系統

這是一個使用 Gradio 建立的音訊轉文字應用程式，支援多種功能：

## 主要功能

- 音訊轉文字（支援 Whisper 和 ElevenLabs）
- 文字優化和摘要生成
- 多語言支援
- Token 使用量和費用計算

## 使用方法

1. 上傳音訊檔案
2. 輸入必要的 API 金鑰
3. 選擇轉錄服務和模型
4. 設定語言選項
5. 點擊處理按鈕

## 安全性說明

- API 金鑰僅在當前處理中使用
- 不會儲存任何敏感資訊
- 每次使用需重新輸入 API 金鑰
